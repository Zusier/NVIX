# NVIX

A fast and portable NVIDIA Driver installer.

## Goals

- [x] Ability to generate *valid* links
- [ ] Download links
- [ ] Run installer
- [ ] Ability to strip components
  - GFE
  - PhysX
  - Audio
  - USB-C Driver
  - Etc.
- [ ] Import and Export presets
- [ ] Fully automated option (no interaction needed with preset provided)
